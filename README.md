# Mã hóa RSA đơn giản 

+ Ứng dụng mã hóa/giải mã sử dụng thuật toán RSA.
+ Sinh viên: Phạm Tuấn Linh.
+ Cài đặt: 
  - Thêm vào file pubspec.yaml
```
      dependecies:
        simple_rsa: ^0.0.5
```
  - Cài đặt package bằng lệnh "flutter pub get".
+ Cách sử dung:
  - Khởi tạo khóa bí mật và khóa công khai:
```
      final PUBLIC_KEY =
      final PRIVATE_KEY =
```
